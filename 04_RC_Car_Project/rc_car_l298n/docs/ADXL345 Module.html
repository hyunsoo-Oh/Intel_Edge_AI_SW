ADXL345 모듈¶

이 프로젝트의 목적은 ADXL345 모듈을 소개하고 Arduino와 Raspberry Pi와 함께 사용하는 방법을 보여드리는 것입니다.

ADXL345 모듈에 대하여¶

  • GND : 전원 공급 장치의 음극.

  • VCC : 전원공급장치는 3-5V에 연결됩니다.

  • CS : 칩 선택.

  • INT1 : 출력 1을 중단합니다.

  • INT2 : 출력 2를 중단합니다.

  • SD0 : 직렬 데이터 출력(SPI 4선), 예비 I2C 주소 선택(I2C).

  • SDA : 직렬 데이터(I2C), 직렬 데이터 입력(SPI 라인 4), 직렬 데이터 입력, 출력(SPI 라인 3)

  • SCL : 직렬 데이터(I2C), 직렬 데이터 입력(SPI 라인 4), 직렬 데이터 입력, 출력(SPI 라인 3)

ADXL345는 작고 얇은 초저전력 3축 가속도계로, 최대 ±16g의 고해상도(13비트) 측정 성능을 제공합니다. 디지털 출력 데이터는 16비트 2의 보수 형식으로 제공되며, SPI(3선 또는 4선) 또는 I2C 디지털 인터페이스를 통해 접근할 수 있습니다.

ADXL345는 모바일 기기 애플리케이션에 매우 적합합니다. 기울기 감지 애플리케이션에서 정적 중력 가속도뿐만 아니라 움직임이나 충격으로 인한 동적 가속도도 측정합니다. 높은 분해능(3.9mg/LSB) 덕분에 1.0° 미만의 경사 변화도 측정할 수 있습니다.

  • ADXL345 - 데이터시트

주요 특징

  • 유형: GY-291

  • 칩: ADXL345

  • 전원 공급 장치: 3~5V

  • 통신 모드: 표준 IIC/SPI 통신 프로토콜

  • 측정 범위: ±2g±16g

  • 3축, ± 2g/ ± 4g/ ± 8g/ ± 16g

애플리케이션

  • 휴대폰

  • 의료기기

  • 게임 및 포인팅 장치

  • 산업용 기기

  • 개인용 내비게이션 장치

  • HDD 낙하 방지

  • 태블릿 컴퓨터 화면 뒤집기

ADXL345의 감지 축

모듈을 엎어놓으면 Z_OUT은 최대값인 +1g가 되고, 엎어놓으면 Z_OUT은 최소값이 됩니다. 어떤 면에 놓이더라도 평평한 표면에 놓이면 X_OUT은 Ax축 방향으로 증가하고, Y_OUT도 Ay축 방향으로 증가합니다. 아래 그림을 참조하세요. 따라서 모듈을 회전시키면 X_OUT, Y_OUT, Z_OUT의 변화를 확인할 수 있습니다.

어떻게 사용하나요?¶

아두이노와 함께 사용하세요¶

Arduino 사용자라면 다음 단계에 따라 모듈을 사용할 수 있습니다.

  1. 회로를 구성합니다.

  2. 라이브러리 관리자 에서 adxl345설치할 라이브러리를 입력 하고 선택하세요. Adafruit ADXL345가 여기에 설치되어 있으며, 팝업 창에서 "모두 설치"를 선택하는 것이 좋습니다.

  3. 샘플 코드를 엽니다.

  4. 보드와 포트를 선택하세요. 여기서는 Arduino Uno이고, 다른 보드를 사용하는 경우 다른 보드를 선택할 수 있습니다.

  5. Arduino 보드에 코드를 업로드합니다.

  6. 시리얼 모니터에서 통신 속도를 9600으로 설정하면 X, Y, Z축의 가속도를 확인할 수 있습니다. 모듈을 사방으로 돌려 출력되는 값이 어떻게 변하는지 확인할 수 있습니다.

라즈베리파이와 함께 사용하세요¶

Raspberry Pi에서 ADXL345 모듈을 사용하려면 다음 튜토리얼을 참조하세요.

1. 라즈베리파이와 ADXL345 모듈 연결

아래 그림을 따라 ADXL345 모듈을 먼저 브레드보드에 연결한 후 라즈베리파이를 연결하세요. 브레드보드가 없어도 암수 케이블을 사용하여 라즈베리파이에 직접 연결할 수 있습니다.

모듈의 GND 핀을 Raspberry Pi의 물리적 핀 6번(GND) 에 연결합니다 . 모듈의 VCC 핀을 Raspberry Pi의 물리적 핀 1번(3v3) 에 연결합니다. 모듈의 SDA 핀을 Raspberry Pi의 물리적 핀 3번(SDA) 에 연결합니다. 모듈의 SCL 핀을 Raspberry Pi의 물리적 핀 5번(SCL) 에 연결합니다.

2. 라즈베리파이 구성하기

Raspberry Pi가 ADXL345 모듈에서 데이터를 검색하려면 먼저 Raspberry Pi의 구성을 변경해야 합니다.

  1. 먼저 다음 두 명령을 실행하여 모든 것이 최신 상태인지 확인해 보겠습니다.

    sudo apt-get update
sudo apt-get upgrade

  2. Raspberry Pi에서 I2C를 활성화하려면 Raspberry Pi를 업데이트한 후 Raspberry 구성 도구를 실행해야 합니다.

    sudo raspi-config

    • 이 화면에서 3개의 인터페이스 옵션 메뉴 로 이동해야 합니다 .

    • 화살표 키를 사용하여 raspi-config 도구 메뉴를 탐색할 수 있습니다. ENTER 키를 눌러 옵션을 선택하세요.

    • 이제 인터페이스 옵션 메뉴에서 I5 I2C를 선택하세요 .

    • ARM I2C 인터페이스를 활성화할지 묻는 메시지가 표시되면 <예>를 선택합니다 .

  3. 이제 I2C가 활성화되었으므로 ADXL345 모듈과 통신하는 데 사용할 패키지를 설치할 수 있습니다. 최신 Raspberry Pi OS(Bullseye)에는 이러한 도구가 기본적으로 설치되어 있으므로 이 단계는 건너뛸 수 있습니다.

    sudo apt-get install python3-dev python3-pip python3-smbus i2c-tools -y

  4. 이제 Raspberry Pi가 ADXL345 모듈을 인식하는지 확인해 보겠습니다.

    sudo i2cdetect -y 1

    이 명령을 실행하면 명령줄에 꽤 많은 정보가 표시되는 것을 볼 수 있습니다. 이 결과에는 적어도 하나의 숫자(예: 53) 가 표시되어야 합니다 .

    pi@raspberrypi:~ $ sudo i2cdetect -y 1

00:                         -- -- -- -- -- -- -- --
10: -- -- -- -- -- -- -- -- -- -- -- -- -- -- -- --

    아무 것도 나타나지 않으면 ADXL345 가속도계가 Raspberry Pi에 제대로 연결되어 있고 센서 핀의 모든 납땜 부위가 깨끗한지 확인하세요. 오류가 발생하면 I2C를 다시 활성화해 보세요.

3. 코드 작성

ADXL345와 통신하려면 Adafruit의 ADXL34x Python 라이브러리를 사용 하지만, 이 라이브러리는 Python 2.x를 지원하지 않으므로 먼저 Python 버전을 확인해야 합니다.

  1. 를 입력하면 python현재 버전의 Python이 나타나고 를 눌러 exit()종료할 수 있습니다.

    pi@raspberrypi:~ $ python
Python 3.9.2 (default, Feb 28 2021, 17:03:44)
[GCC 10.2.1 20210110] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>> exit()

  2. Python 2.x가 있는 경우 다음 명령을 사용하여 Python 3을 설치하세요.

    sudo apt-get install python3

  3. 이제 다음 명령을 사용하여 Adafruit의 ADXL34x Python 라이브러리를 설치할 수 있습니다 .

    sudo pip3 install adafruit-circuitpython-adxl34x

  4. 이제 필요한 모든 패키지가 Raspberry Pi에 설치되었으므로 가속도계에서 정보를 읽는 Python 스크립트를 작성할 수 있습니다. 다음 명령을 사용하여 새 빈 스크립트를 만드세요.

    nano adxl345.py

  5. 다음 샘플 코드를 복사해 넣으세요.

    import time
import board
import adafruit_adxl34x

i2c = board.I2C()  # uses board.SCL and board.SDA

# For ADXL345
accelerometer = adafruit_adxl34x.ADXL345(i2c)

while True:
    print("%f %f %f" % accelerometer.acceleration)
    time.sleep(0.2)

    모든 코드가 정확한지 확인한 후에는 를 누르고 를 눌러 CTRL+X저장 Y하세요 ENTER.

  6. 이제 코드가 완성되었으니 스크립트를 실행해 보겠습니다.

    python3 ./adxl345.py

    이 코드를 실행하면 가속도계에서 값이 읽혀지는 것을 볼 수 있습니다.

4. ADXL345 가속도계 라이브러리에 대한 추가 정보

가속도( )를 읽는 것 외에도 accelerometer.acceleration이 라이브러리는 다음과 같은 감지를 수행할 수도 있습니다.

  • acceleration- x, y 및 z 축의 가속도 값

  • enable_motion_detection- 동작 감지를 활성화합니다. 임계값을 설정할 수 있습니다. 임계값은 기본적으로 18입니다.

  • enable_tap_detection- 탭 감지를 활성화합니다. 단일 또는 이중 탭 감지가 가능합니다.

  • enable_freefall_detection- 자유낙하 감지를 활성화합니다. 임계값과 시간을 설정할 수 있습니다. 임계값은 기본적으로 10이고, 시간은 기본적으로 25입니다.

  • events- 동작 감지, 탭 감지, 자유낙하 감지가 활성화되었을 때 이벤트를 읽는 데 사용됩니다. 읽으려는 이벤트를 지정해야 합니다.

구체적인 사용 예는 https://github.com/adafruit/Adafruit_CircuitPython_ADXL34x/tree/main/examples 에서 확인하세요 .
